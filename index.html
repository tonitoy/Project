Project Requirements

This project required us to use Github for version control while developing this website.

Click here to check out our github and see all the code for the site. The goal of this project is to practice the workflow of social coding, which leads to collaboration within a group on a common set of data. Each team needs to deliever a simple website that is hosted on Github and through Guthub Pages. Each team member needs to create a content block. In our case, the content blocks are the individual cities!

Texas Facts

Texas is a fascinating state. We have our own power grid. We were our own state from 1836 to 1846. We have a privately owned ranch that is larger than Rhode Island. We are known as the Lone Star State. Our official state mammal is an armadillo! These are just some of the fun facts from Texas. Click on one of the cities above to explore even more!
